NOTES, before using the codes:

1) "testexample1.sdb" can be used to test TOCP1 and TOCP2.

2) "testexample2.sdb" can be used to test TOCP3 and TOCP4.

3) In order to use the four codes you need to change the SAP2000 file path located in the main method bellow the Initialization part:

Object.ApplicationStart(SAP2000v15.eUnits.kN_m_C, true, "C:\\...user defined path...\\testexample1.sdb");
(please use \\ between the names of the folders).

4) The corresponding codes for the latest version 19 will also be provided.
